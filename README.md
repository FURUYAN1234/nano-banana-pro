# nano-banana-pro (Super AI 4-koma System)

> **「人間は、どこまで制作から降りられるのか？」**
> AIによるマンガ制作を「自動化」ではなく、その先にある**「無人化」**へと導く実験的Webアプリケーション。

---

## 🚀 概要

本プロジェクトは、人間をクリエイティブな工程から意図的に排除し、AIがディレクターとして「ネタ出し・構成・演出・作画」のすべてを完結させることを目的としています。

## 💡 コンセプト：制作からの「完全離脱」

従来のAIツールが「人間の補助」であったのに対し、本システムは以下の思想に基づいています。

* **トレンド動的同期**: 実行当日の最新ニュースを動的に取得。社会情勢に合わせたネタをAIが自ら選定します。
* **無人ディレクション**: 人間の指示を待たず、AI自身がプロットを組み立て、4コマの構成を決定。
* **同一性の物理的強制**: 独自のプロトコルにより、キャラクター造形を全コマで高精度に維持します。

## 🛠 搭載プロトコル：Super FURU Manga Protocol

作画と演出の制御には、極めて高密度なプロンプトエンジニアリングが採用されています。

### 1. ABSOLUTE PHYSICAL GEOMETRY LOCK

4コマのレイアウト、アスペクト比、パースをピクセル単位で制御。ハルシネーション（AIの誤描画）を物理的に抑制します。

### 2. Weighted Immutable Prompts

キャラクター固有の特徴量（衣装・髪型・小物）に重みを付与。キャラ同士が混ざり合う「フュージョン」を徹底的に防ぎます。

### 3. Kyoto Animation Quality (Level EX-2)

物理ベースのライティング、SSS（肌の透過）、リムライトをエミュレート。最高峰のアニメクオリティを追求します。

## 💻 技術スタック

* **Frontend**: React 19 / Vite 7 / Tailwind CSS
* **LLM/VFM**: Google Gemini API (Nano Banana Pro / 1.5 Pro)
* **API Logic**: 429エラー時の自動リトライおよびモデル切り替え（フォールバック）機構。

## 📝 セットアップ

1. **API Keyの取得**: [Google AI Studio](https://aistudio.google.com/) で Gemini API キーを取得してください。
2. **キーの設定**: アプリ内の設定画面、または `.env` ファイルにAPIキーを入力します。
3. **実行**: `START` ボタンを押すだけで、全自動で漫画が生成されます。
