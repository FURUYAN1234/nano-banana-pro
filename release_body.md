## 🎭 Comedy Structure Engineering / お笑い構造メソッド統合

### What's New / 変更内容

- **[Feature]** Integrated "Comedy Structure Engineering" methodology into both the base scenario generation prompt (STEP2) and the Dialogue/Gag Enhancement prompt (STEP2.5).
- **[Feature]** シナリオ生成（STEP2）とセリフ・ギャグ強化（STEP2.5）の両方に「お笑い構造メソッド」を統合。

### Details / 詳細

The system now explicitly applies structured comedy techniques to produce funnier, more dynamic 4-panel manga scenarios:
4コマ漫画シナリオに構造化されたお笑い技法を明示的に適用し、より面白くダイナミックなシナリオを生成します：

**Gap-Creation Techniques / ズレを作る技法:**
- Substitution (置換) / Exaggeration (誇張) / Role Reversal (逆転)
- Absurdity (不条理) / Tension-Release (緊張と緩和) / Return to Common Sense (常識に戻る)

**Composition Techniques / 構成技法:**
- Setup → Deviation → Escalation → Punchline (フリ→ボケ→溜め→オチ) mapping to 4 panels
- Repetition Escalation (天丼), Riding-Tsukkomi (ノリツッコミ), Stacking (かぶせ), Self-deprecation (自虐)

**Gag Intensity / ギャグ強度:**
- Maximum intensity prioritized. Tame, predictable humor is explicitly prohibited.
- 強烈なボケを最優先。おとなしい優等生ギャグは禁止。
