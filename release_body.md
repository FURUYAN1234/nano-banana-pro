v3.65-alpha: 句読点省略問題の修正 / Fixed punctuation omission issue

## What's New / 更新内容
- OpenAI(ChatGPT)利用時に、セリフの末尾から句読点（。や！）が欠落する問題を修正。
- SystemプロンプトとUserプロンプトの両方から「絶対厳守フォーマット」として句読点の付与を強制（サンドイッチ手法）するように改善。
- 関連ファイルのバージョンをv3.65-alphaへ更新。

- Fixed an issue where OpenAI (ChatGPT) would omit punctuation marks at the end of character dialogue.
- Implemented a "Sandwich Method" by enforcing strict punctuation formatting rules in both System and User prompts to guarantee compliance.
- Updated related file versions to v3.65-alpha.
