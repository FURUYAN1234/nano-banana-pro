## What's New / 変更点

### 🎨 Emotion Style Tag Palette Expansion / 画風パレット拡張

Expanded the Emotion Style Tag system from 17 to 23 styles. AI now automatically selects from a richer palette of visual styles based on scenario context.

AIが文脈から自動選択する画風（Emotion Style Tag）を6種追加し、合計23種に拡張しました。

| New Tag | Style | Use Case |
|---|---|---|
| `THICK_PAINT` | Impasto / 厚塗り | Politics, economy, serious news |
| `PASTEL` | Pastel / パステル | Heartwarming, healing topics |
| `CEL` | Cel Animation / セル画風 | Entertainment, nostalgic topics |
| `DARK_ANIME` | Dark Anime / ダーク調 | Crime, suspense, mysteries |
| `THIN_LINE` | Fine Line Art / 繊細線画 | Emotional, moving moments |
| `HIGH_SATURATION` | Vivid / 高彩度ビビッド | Sports, festivals, excitement |

### 🛡️ Anti-Persona-Gravity Protocol v1.0 / ペルソナ引力抑制

Introduced a new scenario generation rule to prevent authoritative characters (e.g., discipline committee members) from monopolizing 4th-panel punchlines. Punchline ownership is now distributed evenly across all cast members.

特定の権威キャラ（風紀委員等）が4コマ目のオチを毎回独占するパターンを防止する新ルールを導入。全キャラに均等にオチの見せ場が分散されるようになりました。
